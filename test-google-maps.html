üîß Troubleshooting Steps

  1. Get an API Key: Visit Google Maps Platform
  2. Enable APIs: Make sure these APIs are enabled:
    • Maps JavaScript API
    • Geocoding API
    • Directions API
  3. Set up Billing: Google Maps requires billing to be enabled
  4. Check Restrictions: Ensure your domain is allowed (or use no restrictions for testing)
  5. Check Quota: Verify you haven't exceeded your daily quota

üìã Common Error Messages

  • "Oops! Something went wrong" - Usually means API key is invalid or APIs not enabled
  • "REQUEST_DENIED" - API key is invalid or restricted
  • "OVER_QUERY_LIMIT" - Daily quota exceeded
  • "This site cannot be reached" - Network issue or blocked by firewall
